# 🚀 Medium Clone

A modern Medium-inspired blogging platform built using **HTML, CSS, and JavaScript**.  
This project is created to practice frontend development and UI design skills.

---

## 👨‍💻 Author

**Ojash Singh**

---

## 📌 About The Project

This is a simple Medium-style blog homepage clone where users can:
- Browse articles
- Read featured posts
- View staff picks
- Explore trending topics
- Follow suggested developers

The goal of this project is to improve frontend skills by recreating a real-world UI.

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla)

---

## 🎯 Features

- Clean and modern UI design
- Responsive layout
- Interactive buttons (clap, follow, tags)
- Dynamic article rendering using JavaScript
- Sidebar with staff picks & suggestions
- Medium-like layout and typography
